# Slaytabase Privacy Policy
- Last Updated 26 Mar 2026

## Data we collect
The Slaytabase website does not collect any data.
The Slaytabase Discord bot stores logs of any commands used with it for a short while, alongside the user's Discord username.
- These logs are only used to detect bot misuse and find bugs.

The Discord Bot stores information provided to it essential to its function, such as information from usage of the the `/addservermod`, `addusermod`, `/customcommands`, and `/setdiscussionchannel` commands, and full message content when the `<remindme>` command is used. This data includes Server ID or User ID and does not include any other personal information.
